Descripción detallada

Representa el conjunto de características y operaciones relativas al conjunto de especies.

Implementa los métodos para crear la tabla distancias entre cada par de especies, así como las que la mofifican al añadir o eliminar una especie al conjunto (funcionalidades también de este módulo). Ofrece también, el método que iniciliza clústers con la información del conjunto de especies y un método distancia como paso previo para calcular la distancia entre dos especies. A parte, contiene los típicos métodos boolena de existencia y de lectura y escritura. y la consultora para obtener el gen asociado a un identificador. Finalmente, contiene un método para imprimir la tabla de distancias del parámetro implícito.
